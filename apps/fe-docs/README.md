### Prerequisites

-   Node.js >= 18.16.0
-   PNPM: 8.15.8 or 9.14.2

### Development

```bash
# Install dependencies
pnpm install

# Start development server (port 8080)
pnpm dev

# Run linting
pnpm lint

# Run style linting
pnpm stylelint

# Format code
pnpm prettier
```

```bash
# Build for production
pnpm build
```
