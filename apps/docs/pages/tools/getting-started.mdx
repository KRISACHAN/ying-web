# Getting Started

## Installation

You can install `@ying-web/tools` using your preferred package manager:

```bash
# Using pnpm (recommended)
pnpm add @ying-web/tools

# Using npm
npm install @ying-web/tools

# Using yarn
yarn add @ying-web/tools
```

## Basic Usage

```typescript
import { UniversalStorage } from '@ying-web/tools';

// Initialize storage
const storage = new UniversalStorage({
    prefix: 'my-app', // Optional prefix for all keys
    driver: 'local', // 'local' | 'session' | 'memory'
});

// Set a value
storage.set('user', { name: 'John', age: 30 });

// Get a value
const user = storage.get('user');

// Remove a value
storage.remove('user');

// Clear all values
storage.clear();
```

## TypeScript Support

`@ying-web/tools` is written in TypeScript and provides full type definitions out of the box:

```typescript
import { UniversalStorage } from '@ying-web/tools';

interface User {
    name: string;
    age: number;
}

const storage = new UniversalStorage<User>();
storage.set('user', { name: 'John', age: 30 }); // Type-safe!
```
